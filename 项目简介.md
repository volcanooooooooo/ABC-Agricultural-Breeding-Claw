# ABC: Agricultural Breeding Claw — 项目简介

## 项目概述

ABC（Agricultural Breeding Claw）是一款面向育种科学家的 AI 智能助手 Web 应用。用户通过自然语言描述分析需求，系统自动匹配数据集、调用分析工具，完成差异表达分析、GO/KEGG 富集分析、BLAST 序列比对等育种研究任务，并以可视化方式呈现结果。

项目采用前后端分离架构，前端基于 React + TypeScript，后端基于 FastAPI + LangChain Agent。

## 核心功能

| 功能模块 | 说明 |
|---------|------|
| 自然语言对话分析 | 用户以自然语言或 `/analyze` 命令发起分析，LangChain ReAct Agent 自动理解意图并调用对应工具 |
| 差异表达分析 | 基于 t-test / DESeq2 的差异基因筛选，支持火山图可视化 |
| GO/KEGG 富集分析 | 对显著差异基因进行 Gene Ontology 和 KEGG 通路富集，支持本地水稻注释库 |
| BLAST 序列比对 | 上传 FASTA 文件，调用本地 BLAST 数据库进行序列比对 |
| 双轨对比分析 | 同时运行传统统计工具和 LLM 大模型分析，对比两种方法的结果差异 |
| 基因本体可视化 | 基于 React Flow 的本体图谱浏览、编辑与检索 |
| SSE 流式进度 | 分析过程通过 Server-Sent Events 实时推送进度到前端 |

## 技术栈

### 前端

- React 18.2 + TypeScript 5.3
- Vite 5.1（构建工具）
- Ant Design 5.14（UI 组件库）
- React Router v6（路由）
- @xyflow/react 12.0（流程图/本体可视化）
- Recharts 2.12（图表）
- react-markdown + remark-gfm（Markdown 渲染）

### 后端

- FastAPI 0.109 + Uvicorn 0.27
- LangChain + 千问 API（通过 OpenAI SDK 接入）
- Pandas 2.2 / NumPy 1.26 / SciPy 1.12 / scikit-learn 1.4（数据分析）
- pydeseq2 0.4.0（差异表达）
- goatools ≥1.4.0（GO 富集分析）
- SQLAlchemy 2.0（数据库 ORM）
- Pydantic 2.5（数据校验）

## 核心 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/chat/` | POST | 自然语言对话与分析 |
| `/api/datasets` | GET | 获取数据集列表 |
| `/api/analysis/compare` | POST | 双轨对比分析 |
| `/api/analysis/enrichment` | POST | GO/KEGG 富集分析 |
| `/api/analysis/stream/{job_id}` | GET | SSE 流式分析进度 |
| `/api/ontology/` | GET | 获取本体图谱 |
| `/api/blast/upload` | POST | FASTA 文件上传与 BLAST 比对 |

## 数据集

当前主要使用水稻转录组数据集 **GSE242459**：
- 对照组（WT）：DS_WT_rep1/2, N_WT_rep1/2, RE_WT_rep1/2
- 处理组（osbzip23）：DS_osbzip23_rep1/2
- 支持本地水稻基因注释（MH63 品种），包含 GO 和 KEGG 注释映射

## 本地运行

```bash
# 后端
cd backend
pip install -r requirements.txt
PYTHONPATH=backend uvicorn app.main:app --reload --port 8003

# 前端
npm install
npm run dev
```

- 前端：http://localhost:3003
- 后端 API：http://localhost:8003
- API 文档：http://localhost:8003/docs
