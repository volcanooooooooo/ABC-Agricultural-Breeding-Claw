# 育种 AI 科学家系统 - 设计文档

**版本**: 1.0
**日期**: 2026-03-17
**状态**: 已批准

---

## 1. 项目概述

### 1.1 目标

构建一个基于本体的育种 AI 科学家系统，通过自然语言交互完成育种研究中的任务，包括论文研读、自动化数据分析、本体可视化与编辑。

### 1.2 核心价值

- 提高育种研究效率，减少重复性工作
- 降低数据分析门槛，让非技术人员也能进行专业分析
- 统一管理育种知识（本体），提供可视化交互

### 1.3 技术栈

- **后端**: Python FastAPI + LangChain
- **前端**: React + 可视化库
- **本体存储**: JSON 文件（MVP） + Neo4j（扩展）
- **LLM**: 千问在线 API（可配置多模型）
- **部署**: 本地部署

## 3. 功能模块设计

### 3.1 自然语言对话系统

#### 3.1.1 核心流程

```
用户输入自然语言
       ↓
   意图识别 → 解析任务类型（论文研读/数据分析/本体查询）
       ↓
   实体提取 → 提取关键信息（数据集、分析类型、参数）
       ↓
   方案生成 → 展示分析流程（可编辑）
       ↓
   用户确认 → 执行任务
       ↓
   结果展示 → 可视化 + LLM 解读
       ↓
   报告导出
```

#### 3.1.2 意图类型

| 意图 | 示例 | 处理方式 |
|------|------|----------|
| 数据分析 | "分析这批水稻的遗传力" | 跳转分析服务 |
| 本体查询 | "查找基因型相关的条目" | 查询本体库 |
| 论文研读 | "总结这篇论文的方法" | LLM 摘要生成 |
| 系统操作 | "导出分析报告" | 执行导出 |

#### 3.1.3 交互模式

- **自动执行**：简单任务直接执行，实时展示进度
- **交互确认**：复杂任务先展示方案，用户确认后执行
- **多轮对话**：支持上下文连续对话

### 3.2 数据分析模块

#### 3.2.1 支持的数据分析方法

| 类别 | 方法 | 说明 |
|------|------|------|
| 描述性统计 | 均值、方差、分布、分位数 | 基础统计 |
| 相关性分析 | Pearson、Spearman 相关系数 | 变量关系 |
| 遗传力估计 | 狭义遗传力、广义遗传力 | 育种指标 |
| 基因组选择 | GBLUP、RRBLUP | 预测模型 |
| 生信分析 | PCA、聚类、关联分析 | 高级分析 |

#### 3.2.2 数据来源

- **本地文件**: CSV、Excel、TSV
- **数据库**: PostgreSQL、MySQL（预留接口）

#### 3.2.3 分析流程配置

用户通过自然语言描述后，系统自动：
1. 解析数据集路径
2. 选择分析方法
3. 配置参数（可调整）
4. 生成分析流程预览
5. 用户确认后执行

#### 3.2.4 结果展示

- **表格**: 原始数据、统计结果
- **图表**: 折线图、柱状图、散点图、热图、箱线图
- **LLM 解读**: 自动生成结果分析文字
- **报告导出**: PDF、Excel 格式

### 3.3 本体可视化模块

#### 3.3.1 本体内容

| 类别 | 示例 | 属性 |
|------|------|------|
| 基因型 | 水稻品种 "IR8" | 名称、来源、基因组成 |
| 性状 | 株高、穗长、产量 | 定义、单位、测量方法 |
| 代谢组 | 蛋白质、淀粉含量 | 化合物、含量范围 |
| 环境 | 温度、湿度、土壤类型 | 条件、适用范围 |
| 实验方法 | 杂交、筛选、测序 | 步骤、参数、文献 |

#### 3.3.2 可视化设计

- **树形视图**: 类目层级关系，支持展开/折叠
- **关系图**: 力导向图展示实例关联，支持缩放、拖拽
- **搜索**: 关键词高亮定位
- **详情面板**: 点击节点显示属性

#### 3.3.3 编辑功能

- 添加新节点/边
- 编辑节点属性
- 删除节点/边
- 导入/导出本体（JSON 格式）

### 3.4 LLM 配置模块

#### 3.4.1 可配置项

- API 地址（支持千问及兼容 API）
- API Key
- 模型选择
- 温度、Top-P 等生成参数
- 最大 Token 数

#### 3.4.2 配置界面

- 表单式配置
- 连接测试按钮
- 配置保存（本地文件）

## 5. API 设计

### 5.1 核心接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/chat` | POST | 发送对话消息 |
| `/api/ontology/list` | GET | 获取本体列表 |
| `/api/ontology/graph` | GET | 获取本体图数据 |
| `/api/ontology/node` | POST/PUT/DELETE | 节点 CRUD |
| `/api/analysis/run` | POST | 执行分析 |
| `/api/analysis/result/{id}` | GET | 获取分析结果 |
| `/api/config/llm` | GET/POST/PUT | LLM 配置管理 |
| `/api/files/upload` | POST | 文件上传 |
| `/api/export/report` | POST | 导出报告 |
